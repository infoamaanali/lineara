# Lineara: A Simple React component library

A Simple React component library using:

- ⚛️ React + TypeScript
- 🎨 Tailwind CSS v4
- 📦 Rollup bundler

## Availble for

Can be used in Reactjs or Nextjs Applications

## ✅ Installation

```bash
npm install lineara
```
